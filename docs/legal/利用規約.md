# 利用規約

> **本書は法務確認前のドラフト雛形です。施行前に専門家のレビューを受けてください。** 本書の記載は最終・確定したものではなく、法的拘束力を有することを保証するものではありません。「正式版で定めます／記載します」とある箇所は確定前の項目です。

本利用規約（以下「本規約」といいます。）は、Well-Link AI（運営者）（以下「当社」といいます。）が提供する「AI訪問看護記録アシスト」（以下「本サービス」といいます。）の利用条件を定めるものです。

---

## 第1条（適用）

1. 本規約は、本サービスの提供および利用に関する当社と利用者との間の一切の関係に適用されます。
2. 当社が本サービス上または当社ウェブサイト上で別途定める各種規定・ガイドライン等は、本規約の一部を構成します。本規約と当該規定等の内容が矛盾する場合は、当該規定等が優先して適用されるものとします。
3. 本サービスは、訪問看護ステーション等の事業所による業務利用を前提とした B2B サービスです。

## 第2条（定義）

本規約において、次の各号に掲げる用語の意味は、当該各号に定めるところによります。

1. 「利用契約者」とは、本サービスの利用について当社と契約を締結する訪問看護ステーション等の事業所をいいます。
2. 「利用者」とは、利用契約者に所属し、利用契約者の管理のもとで本サービスを利用する個人（看護師・管理者等）をいいます。
3. 「対象者」とは、利用契約者が提供する訪問看護サービスを受ける個人（患者・要介護者等）およびその家族をいいます。
4. 「医療・介護情報」とは、対象者の年齢・主病名・要介護度、SOAP記録、看護計画書・各種報告書その他本サービス上で取り扱われる対象者に関する情報をいいます。
5. 「生成AI」とは、本サービスが処理のために利用する Anthropic 社（米国の事業者）の大規模言語モデル「Claude」（API経由）その他の生成AIをいいます。

## 第3条（利用登録・アカウント）

1. 本サービスの利用を希望する事業所は、当社所定の方法により利用登録を申請し、当社がこれを承認することにより利用契約が成立します。
2. 本サービスのアカウント認証には Google アカウント（Google OAuth）を利用します。利用者は、自己の責任において認証情報を管理するものとします。
3. 本サービスは事業所ごとにデータを分離して管理します（行レベルセキュリティ（RLS）による分離）。利用契約者は、自らの事業所に所属する利用者にのみ適切な権限を付与し、アカウントを管理するものとします。
4. アカウントの管理不十分・第三者の使用・盗用等による損害の責任は利用契約者が負うものとし、当社は責任を負いません。

## 第4条（事業所・利用者の責任）

1. 利用契約者および利用者は、関係法令（個人情報の保護に関する法律、医療・介護関連法令等）および本規約を遵守して本サービスを利用するものとします。
2. **対象者本人および家族への説明ならびに同意の取得は、利用契約者の責任において行うものとします。** 当社は、対象者本人・家族との直接の契約関係を有しません。
3. 利用契約者は、本サービスにおいて医療・介護情報および後記第6条に定めるAIへの送信・第三者提供が行われることについて、対象者本人または家族から、自らの責任で必要な同意を取得するものとします。
4. 利用契約者は、対象者の氏名・住所・生年月日等の特定の個人を識別できる情報を、当社の推奨する運用（個人情報をマスクのうえ撮影する等）に従い、生成AIへ送信しないよう努めるものとします。

## 第5条（禁止事項）

利用者は、本サービスの利用にあたり、次の各号に掲げる行為をしてはなりません。

1. 法令または公序良俗に違反する行為
2. 当社、他の利用者または第三者の権利・利益を侵害する行為
3. 本サービスの運営を妨害する行為、不正アクセス、リバースエンジニアリング等
4. 第4条に反して必要な同意を得ないまま対象者情報を入力・送信する行為
5. 生成AIの出力を看護師による確認・修正を経ずにそのまま正式な記録・判断として用いる行為
6. 本サービスを利用契約に定める目的の範囲を超えて利用する行為
7. その他、当社が不適切と合理的に判断する行為

## 第6条（医療情報・個人情報の取扱い：AI送信・第三者提供への同意）

1. 本サービスは、利用者が入力したテキスト（SOAP記録等の医療情報の一部）および画像を、処理のため生成AI（Anthropic 社「Claude」、API経由、米国の事業者）へ送信し、当該事業者の環境で処理します。これは、個人情報保護法上の**第三者提供、外国にある第三者への提供および外部委託に該当しうる**取扱いです。
2. 利用契約者は、前項のAIへの送信および国外（米国）への移転を含む取扱いについて、第4条に基づき対象者本人または家族から必要な同意を取得したうえで本サービスを利用するものとします。
3. Anthropic 社は、原則として API 経由で送信されたデータをモデルの学習に使用せず、一定期間（最大30日程度）のみ保持するとされています。ただし、これらの前提は同社の契約・規約により別途確認のうえ確定するものとし、当社はその内容を保証するものではありません。
4. 当社は、対象者の氏名・住所・生年月日等の個人を識別する情報を生成AIへ送信しない運用方針を採用しています。ケアプランは個人情報をマスクして撮影する等の運用を推奨します。
5. 当社は、医療情報システムの安全管理に関する3省2ガイドライン（厚生労働省「医療情報システムの安全管理に関するガイドライン第6.0版」等）に準拠することを目指して本サービスを運営します。

## 第7条（データの保存・バックアップ・消去）

1. 医療・介護情報を含む本サービスのデータは、クラウドデータベース（Supabase / PostgreSQL、AWS 東京リージョン上でのホスティングを前提とします。）に保存されます。
2. 当社は、合理的な範囲でデータのバックアップを行いますが、データの完全性・復元を保証するものではありません。利用契約者は、必要に応じて自らバックアップを行うものとします。
3. 利用契約の終了後、当社は当社所定の期間経過後にデータを消去することができます。データの保存期間および消去の詳細はプライバシーポリシーに定めるところによります。

## 第8条（料金）

1. 本サービスは、現在、試験提供期間として無償で提供されます。
2. 当社は、将来、本サービスの全部または一部を有償化することがあります。その場合、当社は料金体系・課金条件を別途定め、相当な期間をもって利用契約者に周知するものとします。
3. 有償化後の料金、支払方法、支払期日その他の条件は、当社が別途定める料金規定（有償化時に利用契約者へ提示します）によるものとします。

## 第9条（免責・非保証）

1. **本サービスの生成AIによる出力は、あくまで下書き（ドラフト）であり、看護師による内容の確認・修正が必須です。** 利用者は、出力をそのまま正式な記録・判断として用いてはなりません。
2. **医療・看護に関する判断および記録の最終的な責任は、利用契約者および当該記録を作成・確認する看護師にあります。** 当社は、生成AIの出力の正確性・完全性・適切性・特定目的への適合性を保証しません。
3. 当社は、本サービスの提供にあたり、可用性・無謬性・中断のないことを保証しません。
4. 当社は、本サービスの利用または利用不能から生じた利用契約者・利用者・対象者その他の第三者の損害について、当社の故意または重大な過失による場合を除き、責任を負わないものとします。
5. 当社が損害賠償責任を負う場合であっても、その範囲は通常生ずべき直接かつ現実の損害に限られ、当社が別途定める上限額（正式版で確定します）を超えないものとします。

## 第10条（サービスの変更・中断・終了）

1. 当社は、利用契約者に事前に通知することなく、本サービスの内容を変更し、または提供を中断することができます。ただし、重大な変更については相当な期間をもって周知するよう努めます。
2. 当社は、システムの保守・障害・天災その他やむを得ない事由により、本サービスの全部または一部の提供を一時的に中断することができます。
3. 当社は、相当な予告期間をもって、本サービスの提供を終了することができます。

## 第11条（秘密保持）

1. 当社および利用契約者は、本サービスの利用に関連して知り得た相手方の営業上・技術上の情報および医療・介護情報を、相手方の事前の書面による承諾なく第三者に開示・漏えいせず、本サービスの利用・提供の目的以外に使用しないものとします。
2. ただし、本サービスの提供に必要な範囲で第6条・プライバシーポリシーに定める委託先・提供先に情報を取り扱わせる場合は、この限りではありません。

## 第12条（準拠法・裁判管轄）

1. 本規約の準拠法は日本法とします。
2. 本サービスまたは本規約に関して当社と利用契約者・利用者との間に生じた紛争については、当社所在地を管轄する地方裁判所（正式版で確定します）を第一審の専属的合意管轄裁判所とします。
