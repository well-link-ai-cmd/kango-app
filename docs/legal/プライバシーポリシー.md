# プライバシーポリシー

> **本書は法務確認前のドラフト雛形です。施行前に専門家のレビューを受けてください。** 本書の記載は最終・確定したものではなく、法的拘束力を有することを保証するものではありません。「正式版で定めます／記載します」とある箇所は確定前の項目です。

Well-Link AI（運営者）（以下「当社」といいます。）は、「AI訪問看護記録アシスト」（以下「本サービス」といいます。）の提供にあたり、利用者および対象者の情報を以下のとおり取り扱います。本サービスは訪問看護ステーション等の事業所による業務利用を前提としています。

---

## 1. 事業者情報

- 事業者名：Well-Link AI（運営者）
- 所在地：正式版で記載します
- 代表者：正式版で記載します
- 個人情報保護に関する問い合わせ先：well-link-ai@05company.com（本ポリシー第10項に記載）

## 2. 取得する情報

当社は、本サービスの提供にあたり、次の情報を取得します。

1. **アカウント情報**：Google アカウント（Google OAuth）に紐づくメールアドレス、表示名等の認証関連情報、所属事業所・権限等。
2. **医療・介護情報**：対象者の年齢・主病名・要介護度、SOAP記録、看護計画書、各種報告書（月次報告書・情報提供書等）その他利用者が本サービスに入力する対象者に関する情報。
3. **画像**：ケアプラン等を撮影した画像（個人情報をマスクのうえ撮影する運用を推奨）。
4. **アクセスログ・監査ログ**：アクセス日時、操作内容、IPアドレス、利用環境（ブラウザ・端末情報）等。

## 3. 利用目的

当社は、取得した情報を次の目的のために利用します。

1. 本サービスの提供・運営・本人認証・事業所ごとのデータ分離管理
2. SOAP記録・看護計画書・各種報告書等の作成補助（生成AIによる下書き生成を含む）
3. 本サービスの保守・障害対応・品質改善・不正利用の防止
4. 利用者への連絡、問い合わせ対応
5. 法令に基づく対応その他上記に付随する目的

## 4. 外部送信・第三者提供・委託（国外移転への同意を含む）

1. 当社は、本サービスの提供に必要な範囲で、次の外部事業者に情報の取扱いを委託し、または提供します。

   | 委託先・提供先 | 用途 | 所在 |
   |---|---|---|
   | Anthropic 社（生成AI「Claude」、API経由） | 入力テキスト・画像のAI処理（記録生成等） | 米国 |
   | Google 社（生成AI「Gemini」、API経由） | お問い合わせ内容の自動整理（分類・要約・返信下書き）※利用する場合 | 米国 |
   | Google 社（Google Apps Script／Gmail） | お問い合わせの通知・自動受付返信メールの送信 | 米国 |
   | Supabase（データベース／認証基盤） | データの保存・認証 | AWS 東京リージョン前提 |
   | Vercel（アプリケーションホスティング） | 本サービスの配信 | 米国 |
   | Supabase（データベース・認証基盤） | 記録データの保存・ログイン認証 | 米国等 |
   | Anthropic（AI処理） | 記録ドラフトの生成（氏名・住所等は送信しない運用） | 米国 |
   | Google（ログイン認証・問い合わせ通知） | Googleアカウント認証・通知メール | 米国等 |

   ※ お問い合わせ内容を生成AI（Gemini）で処理する場合があります。**お問い合わせ本文には患者等の個人情報を記載しないでください**（無料版の生成AIでは入力がモデル改善に利用される可能性があるため）。

2. Anthropic 社への送信は**外国（米国）にある第三者への提供**に該当しうるため、当社は、利用契約者を通じて対象者本人または家族の同意を取得したうえで本サービスを提供します。利用契約者は、当該国外移転を含む取扱いについて必要な同意を取得する責任を負います（利用規約第4条・第6条）。
3. 当社は、法令に基づく場合を除き、本項に定める委託先・提供先以外の第三者に情報を提供しません。

## 5. AIによる処理と学習不使用の前提

1. 本サービスは、利用者が入力したテキスト（SOAP記録等の医療情報の一部）および画像を、処理のため生成AI（Anthropic 社「Claude」）へ送信します。
2. Anthropic 社は、原則として API 経由で送信されたデータをモデルの学習に使用せず、一定期間（最大30日程度）のみ保持するとされています。ただし、これらの前提は同社の契約・規約により別途確認のうえ確定するものとし、当社がその内容を保証するものではありません。
3. 当社は、対象者の氏名・住所・生年月日等の特定の個人を識別する情報を生成AIへ送信しない運用方針を採用しています。ケアプランは個人情報をマスクして撮影する等の運用を推奨します。

## 6. 安全管理措置

当社は、取得した情報の漏えい・滅失・毀損の防止その他安全管理のため、次の措置を講じます。

1. **暗号化**：通信時の暗号化（TLS）および保存時の暗号化。
2. **アクセス制御**：事業所ごとのデータ分離（行レベルセキュリティ／RLS）、権限管理、認証（Google OAuth）。
3. **監査ログ**：アクセス・操作の記録および監視。
4. **委託先の監督**：委託先・提供先に対し、本ポリシーおよび関係法令に沿った適切な取扱いを求め、必要な監督を行います。
5. 当社は、医療情報システムの安全管理に関する3省2ガイドライン（厚生労働省「医療情報システムの安全管理に関するガイドライン第6.0版」等）に準拠することを目指します。

## 7. 保有期間と消去

1. 当社は、利用目的の達成に必要な期間、情報を保有します。
2. 利用契約の終了後、当社所定の期間（正式版で定めます）の経過後に、当社はデータを消去または利用できない状態にします。
3. 生成AIへ送信されたデータの保持期間は、第5項に定める委託先の規約によります。

## 8. 利用者（本人）の権利と開示請求の窓口

1. 対象者本人は、自己の個人情報について、利用目的の通知、開示、訂正・追加・削除、利用停止・消去、第三者提供の停止等を求めることができます。
2. 本サービスにおける対象者の情報は、利用契約者（訪問看護ステーション）の管理下で取り扱われます。**対象者本人からの開示等の請求は、第一次的には利用契約者が対応する**ものとし、当社は利用契約者と連携してこれに対応します。
3. 開示等の請求の窓口は、第10項に定める問い合わせ窓口とします。

本サービスは、ログインセッションの維持・認証のために Cookie および類似技術を使用します。これらは本サービスの提供に必要な範囲で利用し、認証セッションの管理を目的とします。
