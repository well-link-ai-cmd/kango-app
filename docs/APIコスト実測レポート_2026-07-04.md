# kango-app APIコスト実測レポート（価格再提案の基礎資料）

- 実測日: 2026-07-04
- 実測方法: `tests/prompts/soap/run.ts soap all`（全10ケース・Claude Haiku 4.5・本番プロンプト同一 promptHash=cdaecf0b）
- 単価: Haiku 4.5 = 入力 $1.00/M・出力 $5.00/M・キャッシュ読取 $0.10/M・キャッシュ書込(1h TTL) $2.00/M
- 為替前提: **1USD = 150円**（変動時は比例換算）

## 1. 実測生データ（SOAP生成・10ケース）

| 項目 | 実測値 |
|------|--------|
| 固定system（キャッシュ対象） | **14,097トークン**（全コール共通） |
| 可変入力 | 669〜1,120（平均 **908**） |
| 出力 | 479〜1,276（平均 **890**） |
| 応答時間 | 4.1〜9.3秒 |
| キャッシュ動作 | 全10ケースで cache_read=14,097（ヒット率100%・1時間TTL内） |

## 2. SOAP生成 1コールあたりコスト（キャッシュ有無比較）

| 状態 | 内訳 | USD | 円換算 |
|------|------|-----|--------|
| **キャッシュヒット時** | 読取14,097×$0.1 + 入力908×$1 + 出力890×$5 | $0.0068 | **約1.0円** |
| **キャッシュミス時**（1時間TTL書込） | 書込14,097×$2 + 入力908×$1 + 出力890×$5 | $0.0336 | **約5.0円** |
| （参考）キャッシュ無しの場合 | 入力15,005×$1 + 出力890×$5 | $0.0195 | 約2.9円 |

- ポイント: 1時間TTLの書込は2倍課金のため、**ミス時は無キャッシュより高い**。1時間以内に3コール以上（＝朝のまとめ書き・複数スタッフ運用）で逆転し、ヒット時は無キャッシュ比 **65%減**
- 確認質問（questions）ルート: systemが2,048トークン未満でキャッシュ対象外。入出力とも小規模で **1コール約0.6円（推定・未実測）**

**→ 1記録あたり（SOAP生成＋確認質問）: ヒット時 約1.6円 / ミス時 約5.6円**

## 3. 月間コスト試算（SOAP系＋周辺ルート込み）

前提: スタッフ1人あたり1日4件・月22日稼働。周辺ルート（月次報告書・看護計画・褥瘡計画等）は低頻度で月200〜300円と概算。

| シナリオ | 件数/月 | キャッシュヒット率 | 月額試算 |
|---------|--------|-----------------|---------|
| A. 現状（1人試験運用・散発） | 約90件 | 50% | **約600〜800円** |
| B. 全員利用（5人・朝まとめ書き） | 約440件 | 80% | **約1,300〜1,600円** |
| C. 全員利用（悲観・散発利用でヒット率低） | 約440件 | 30% | **約2,200〜2,400円** |

### 実績との突合

- 2026年5月実績: **$8.46 ≈ 1,270円**（1人試験運用・ただしPrompt Caching導入は5/29のため**ほぼ非キャッシュ期間**＋開発テスト分を含む）
- シナリオAが5月実績より安いのはキャッシュ効果＋テスト分除外のため。**6月のAnthropic Console実績（キャッシュ有効の初のフル月）との突合を推奨**（次回アクション）

## 4. 価格再提案への示唆

- 1事業所（スタッフ5人規模）のAPI原価は**月1,300〜2,400円**。利用者1人あたりに直すと**月14〜49円**（週2訪問想定）
- ベーシック月5,000円で提供してもAPI原価率は26〜48%（悲観ケースでも粗利あり）。月1万円なら13〜24%
- **結論: APIコストは価格決定の制約にならない。価格は原価積み上げでなく価値ベース（記録時間削減・記録の質）で設定可能**
- ※価格の最終決定は🔴オーナー判断。本レポートは判断材料の提供のみ

## 5. 留意点・未実測事項

1. **questionsルートは推定値**（run.tsのquestionsモードにトークン表示がないため。必要なら次回改修で実測化）
2. テストケースの入力長は典型例ベース。実運用の長文メモではコール単価が1〜2割上振れしうる
3. マルチテナント展開時はSupabase等インフラ費が別途発生（本レポートはAnthropic APIのみ）
4. モデル単価改定・為替変動でスライドする。四半期ごとに再確認
5. hikitsugi作業リスト4「確認質問・アラートの上位モデルA/B（Haiku vs Sonnet 5）」を実施した場合、questionsルートは単価3倍（Sonnet $3/M入力）だが件数が少なく月額影響は+100〜300円程度と見込む
